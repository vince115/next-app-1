# 專案概述

該專案涉及創建一個在行動裝置、平板電腦和桌面裝置上看起來很棒的響應式網站。此設計可流暢地適應不同的螢幕尺寸，無論使用何種設備，都能提供最佳的使用者體驗。

## 特徵

- 行動優先設計
- 用於佈局的 CSS Flexbox 和 Grid
- 跨瀏覽器相容性
